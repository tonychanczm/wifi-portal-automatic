# あるスキー場のWIFIロクイン自動化ツール

😇スキー場さん、30分ごとに回線を切断する仕様は、素晴らしいですね！

お陰様で、技術は革命を余儀なくされました。

WIFIのロクインAPIは謎の暗語化仕様で、アルゴリズムの解析などせず、AIで以下のNode.jsスクリプトを作成しました：

1. 存在しないURLを叩く→通ったらWIFIのキャプティポータルに優しく捕獲された判定
2. ロクインページに飛ばすターゲットURLを正規表現で抽出
3. playwrightというライブラリーでヘッドレスChromiumを操作して、人間ごっこしてWIFIをログイン！
4. おわり👍

ありがとうキャプティポータル、今日も文明はは一歩前進しました

## ライブラリーのインストール

```bash
pnpm install
```

## Chromiumのインストール

```bash
pnpm exec playwright install chromium
```
## 実行

```bash
pnpm start
```
